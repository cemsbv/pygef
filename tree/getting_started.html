Getting started

Getting started with pygef is easy done by importing the pygef library:

In [1]: from pygef import Cpt, Bore

or any equivalent import statement.

Load a Cpt/Bore file

The classes Cpt and Bore accept two possible inputs:

  • the path of the file

  • the content, a dictionary containing the keys: [“string”, “file_type”]

    • string: String version of the file.

    • file_type: One of [“gef”, “xml”]

If you want to use the path then your code should look like this:

In [2]: import os

In [3]: path_cpt = os.path.join(os.environ.get("DOC_PATH"), "../pygef/test_files/cpt.gef")

In [4]: cpt = Cpt(path_cpt)

If you want to use the content method:

In [5]: with open(path_cpt, encoding="utf-8", errors="ignore") as f:
   ...:     s = f.read()
   ...: 

In [6]: gef = Cpt(content=dict(string=s, file_type="gef"))

Access the attributes

Accessing the attributes of a pygef object is quite easy. If for example we want to know the (x, y, z) coordinates of the gef we can simply do:

In [7]: coordinates = (gef.x, gef.y, gef.zid)

In [8]: print(coordinates)
(79578.38, 424838.97, -0.09)

Check all the available attributes in the reference. Everything(or almost) that is contained in the files it is now accessible as attribute of the gef object.

The classes Cpt and Bore have different attributes, check the reference to learn more about it.

A common and very useful attribute is gef.df, this is a polars.DataFrame that contains all the rows and columns defined in the file.

Classify a cpt

We can classify a cpt object using the method .classify(). The available classification are:

  • Robertson(1990) and Robertson(2016)

  • Jefferies&Been: available only if u2 is in the columns of cpt.df.

It is possible to use the old (1990) or new(2006) implementation of Robertson.

Robertson(1990):

\[\begin{align} I_c = \sqrt{(3.47 - \log Q_t )^2 + (\log F_r + 1.22)^2} \end{align}\]

where:

\[\begin{split}\begin{align} Q_t &= \frac{q_t - \sigma_{vo}}{\sigma_{vo}'}, \\ F_r &= \frac{f_s}{q_t - \sigma_{vo}} \times 100, \\ \end{align}\end{split}\]
\[\begin{split}\begin{align} q_t = \begin{cases} q_c + u_2 (1 - a), & \text{if} \> u_2 \neq 0 \\ q_c & \text{otherwise} \end{cases} \end{align}\end{split}\]

Robertson(2006):

\[\begin{align} I_c = \sqrt{(3.47 - \log Q_t )^2 + (\log F_r + 1.22)^2} \end{align}\]

where:

\[\begin{split}\begin{align} Q_t &= \frac{q_t - \sigma_{vo}}{p_a}(\frac{p_a}{\sigma_{vo}'})^n, \\ n &= 0.381 \times I_c + 0.05 \frac{\sigma_{vo}'}{p_a} - 0.15, \\ F_r &= \frac{f_s}{q_t - \sigma_{vo}} \times 100, \\ \end{align}\end{split}\]
\[\begin{split}\begin{align} q_t = \begin{cases} q_c + u_2 (1 - a), & \text{if} \> u_2 \neq 0 \\ q_c & \text{otherwise} \end{cases} \end{align}\end{split}\]

The implementation given by Jefferies&Been is the following:

\[\begin{align} I_c = \sqrt{(3 - \log (Q_t \times(1 - u_2) + 1))^2 + (1.5 +1.3 \times \log F_r)^2} \end{align}\]

where:

\[\begin{split}\begin{align} Q_t &= \frac{q_t - \sigma_{vo}}{\sigma_{vo}'}, \\ F_r &= \frac{f_s}{q_t - \sigma_{vo}} \times 100, \\ q_t &= q_c + u_2 (1 - a) \end{align}\end{split}\]

The classification is done for each row of the cpt, and you can get the result for each row.

Check the reference to learn about all the arguments of the method.
